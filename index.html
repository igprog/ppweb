O aplikaciji


Program Prehrane Web je web aplikacija za izradu raznovrsnih i uravnoteženih jelovnika i plana tjelesne aktivnosti.


Web aplikacija Program Prehrane Web nasljednica je višestruko nagrađivanog računalnog programa Program Prehrane 5.0, a nastala je iz potrebe za izradom alata koji će nutricionistima i zdravstvenim djelatnicima ali i svima ostalima koji vode računa zdravoj i uravnoteženoj prehrani omogućiti brzu i jednostavnu izradu jelovnika te plana tjelesne aktivnosti.

Aplikacija Program Prehrane Web omogućava utvrđivanje stanja uhranjenosti i energetske potrošnje pojedinca te izradu jelovnika utemeljenu na znanstvenim spoznajama na polju nutriconizma. Osmišljavanjem jelovnika korištenjem ove aplikacije, prehrana krajnjih korisnika postaje uravnotežena i raznolika što se može zahvaliti činjenici da se izrada jelovnika bazira na preporučenom broju jediničnih serviranja svih skupina namirnica te njihovih energetskih, makro i mikronutiritivnih vrijednost. S obzirom da web aplikacija Program Prehrane Web sadrži bazu od preko 660 namirnica i jela moguće je zadovoljiti zahtjeve većine korisnika po pitanju izbora namirnica. Osim toga aplikacija pruža mogućnost unosa vlastitih namirnica i njihovih makro i mikronutirtivnih vrijednosti. Nadalje, kad se govori o problemu prekomjerne tjelesne mase, preporuča se uz promjenu prehrambenih navika, u svakodnevicu uključiti i dodatnu tjelesnu aktivnost, pa tako Program Prehrane Web nudi podatke o energetskoj potrošnji za preko 400 tjelesnih aktivnosti.
